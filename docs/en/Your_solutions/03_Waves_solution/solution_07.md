# Task 07 – Standing Wave Modes

## Problem Statement

A standing wave with four antinodes is produced on a string of length $L = 80 \,\text{cm} = 0.80 \,\text{m}$. Find the wavelength.

## Theory

For a string fixed at both ends, a mode with $n$ antinodes satisfies

$$
L = n \frac{\lambda}{2}
$$

Hence,

$$
\lambda = \frac{2L}{n}
$$

## Step-by-Step Solution

There are four antinodes, so $n = 4$. Therefore,

$$
\lambda = \frac{2L}{4}
$$

Substitute $L = 0.80 \,\text{m}$:

$$
\lambda = \frac{2(0.80)}{4}
$$

$$
\lambda = 0.40 \,\text{m}
$$

## Final Result

$$
\lambda = 0.40 \,\text{m}
$$

## Interpretation

Each antinode occupies half a wavelength segment. Four antinodes over the string length imply that the full wavelength is $0.40 \,\text{m}$.
